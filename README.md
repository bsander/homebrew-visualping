# Homebrew Tap for visualping

This branch contains the Homebrew formula for [visualping](https://github.com/bsander/homebrew-visualping/tree/main).

## Install

```sh
brew tap bsander/visualping
brew install visualping
```

## Development

Source code and development happens on the [`main` branch](https://github.com/bsander/homebrew-visualping/tree/main).
